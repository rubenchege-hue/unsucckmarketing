SEO in 2026 isn't just about keywords anymore. It's about **E-E-A-T, Core Web Vitals, and semantic relevance**. Here's our complete audit checklist.

## Technical SEO

### Crawlability & Indexability

- [ ] XML sitemap is submitted and error-free
- [ ] Robots.txt isn't blocking important pages
- [ ] No orphan pages (pages with no internal links)
- [ ] Canonical tags are correctly implemented
- [ ] 404 pages return proper status codes

```bash
# Quick crawl check using curl
curl -s -o /dev/null -w "%{http_code}" https://yoursite.com/page
```

### Core Web Vitals

Google's page experience signals are more important than ever. **Aim for:**

| Metric | Target | Current Avg |
|--------|--------|-------------|
| LCP (Largest Contentful Paint) | < 2.5s | 3.1s |
| FID (First Input Delay) | < 100ms | 85ms |
| CLS (Cumulative Layout Shift) | < 0.1 | 0.15 |

## On-Page SEO

### Title Tags & Meta Descriptions

Every page should have a unique, compelling title tag under 60 characters and a meta description under 160 characters.

**The formula:**
```
[Primary Keyword] + [Unique Value Prop] | [Brand Name]
```

### Content Quality

- [ ] Content matches search intent (not just keyword match)
- [ ] Headings follow a proper H1 → H2 → H3 hierarchy
- [ ] Internal links point to relevant, deep content
- [ ] External links go to authoritative sources
- [ ] Image alt text is descriptive, not keyword-stuffed

> "Google's helpful content update rewards content written for people, not search engines. Write for humans first, optimize for search second."

## Content Audit

### Identify Content Gaps

Compare your content against what your competitors rank for. **Tools we use:**

1. **Ahrefs / SEMrush** — Keyword gap analysis
2. **Google Search Console** — Actual search performance data
3. **AnswerThePublic** — Question-based content ideas
4. **Your own CRM** — What questions do prospects ask?

### Content Refresh Opportunities

Don't just create new content — update existing high-performing pieces. A content refresh can boost rankings by **30-50%** with a fraction of the effort.

## Off-Page SEO

- [ ] Backlink profile is clean (no toxic links)
- [ ] Competitor backlinks are analyzed for opportunities
- [ ] Brand mentions are tracked and converted to links
- [ ] Local citations are consistent (Google Business Profile, etc.)

## The 80/20 Action Plan

Most businesses can get **80% of the results from 20% of the fixes**. Focus on:

1. Fix critical technical issues (crawlability, speed)
2. Optimize pages that already rank on page 2-3
3. Fill content gaps for high-intent keywords
4. Build 2-3 quality backlinks per month
